# z-log

> [Zguillez](https://zguillez.io) | Guillermo de la Iglesia

## Helper for console.log()

# Getting Started

## Install

```
npm i z-log --save-dev
yarn add z-log --dev
```

# Usage

```
const zlog = require('z-log');
```

## Print console.logs

```
zlog.log('test log');
zlog.success('success log');
zlog.warning('warning log');
zlog.info('info log');
zlog.error('error log');
```
